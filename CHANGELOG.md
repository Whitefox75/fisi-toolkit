# Changelog

Alle wichtigen Änderungen an diesem Projekt werden in dieser Datei dokumentiert.

## [1.0.0] - 2026-02-13

### Hinzugefügt
- **Netzwerk-Tab**: IP/Subnetz-Rechner mit binärer Visualisierung
- **Speicher-Tab**: RAID-Rechner (RAID 0, 1, 5, 6, 10)
- **Logik-Tab**: 32-Bit Matrix für Hex/Dez/Bin-Konvertierung
- **Einheiten-Rechner**: Binäre und dezimale Speichereinheiten
- **Settings-Tab**: Design-Modi und UI-Skalierung
- **Info-Tab**: Credits und Social-Media-Links
- **Kollabierbare Sidebar**: Platzsparendes Design
- **Clipboard-Integration**: Kopiere Ergebnisse mit einem Klick
- **Dark/Light Mode**: Automatische Systemerkennung
- **PyInstaller Support**: Erstelle Standalone-EXE

### Technisch
- CustomTkinter für modernes UI
- Responsive Layout mit Grid/Pack
- Input-Validierung für alle Berechnungen
- Fehlerbehandlung für ungültige Eingaben
